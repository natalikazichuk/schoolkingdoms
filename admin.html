🗺 Карта королівства

Ступінь → клас → предмет → розділ → тести. Саме це бачить дитина на сторінці «Тести». Тест потрапляє в розділ, якщо його обрано у списку нижче або якщо в тесті заповнено поле «Розділ (topic)» такою самою назвою.
🔗 Ланцюг — тести розділу відкриваються по черзі, кожен наступний після 55% попереднього. Розділи з однаковою групою ланцюга зчіплюються в один спільний ланцюг.

Імпорт тестів (JSON)

Встав один тест або масив тестів. Звичайний: { title, subject, grade, stat, …, questions:[{ q, options:[…], correct }] }
Рівні: { title, …, format:"levels", levels:[{ level, name, questions:[…] }] }
Ілюстрація (необов'язково, у питанні): "visual": { "type":"number-line", "from":1, "to":10, "highlight":[4] } або { "type":"number-blocks", "value":6, "groups":[2,4] }
